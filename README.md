# Treinamentos Sienge (GitHub Pages + Jekyll)

## Implementado
- **Ordenação por módulo** via campo `order` em `_videos/*.md`
- **Selo NOVO / ATUALIZADO** via campo `status`
- Tema alternável (Dark suave / Claro)

## Como usar
- Publique com GitHub Pages (branch `main`, pasta root)
- Edite/adicione arquivos em `_videos/`

### Campos úteis
- `order`: número (10, 20, 30...) para ordenar dentro do módulo
- `status`: `novo` ou `atualizado`
